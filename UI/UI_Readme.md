# UI配置文件说明

本文档说明如何修改登录页面的显示内容。

---

## 文件位置

所有可配置的文本内容都在 `UI/login_config.json` 文件中。

---

## 配置字段说明

### 1. 平台信息 (platform)

| 字段 | 说明 | 示例 |
|------|------|------|
| `name` | 平台主标题 | 儿童先心病智能诊断平台 |
| `subtitle` | 副标题 | AI辅助先天性心脏病超声诊断 |
| `tagline` | 大标题第一行 | 精准、高效、智能 |
| `tagline2` | 大标题第二行 | 先心病辅助诊断工具 |

### 2. 三栏介绍 (features)

数组形式，包含3个元素，分别对应三个介绍栏。

| 字段 | 说明 | 示例 |
|------|------|------|
| `title` | 栏目标题 | AI智能分析 |
| `description` | 栏目标述 | 采用深度学习模型自动识别... |

### 3. 按钮文字 (buttons)

| 字段 | 说明 | 示例 |
|------|------|------|
| `topRight` | 右上角按钮 | 医师入口 |
| `titleBelow` | 中间按钮 | 点击登录 |

---

## 修改示例

### 示例1：修改平台名称

```json
{
  "platform": {
    "name": "我的诊断平台",
    "subtitle": "AI辅助诊断系统"
  }
}
```

### 示例2：修改三栏介绍内容

```json
{
  "features": [
    {
      "title": "智能识别",
      "description": "使用AI技术自动识别医学影像特征。"
    },
    {
      "title": "多模态显示",
      "description": "支持多种视图模式切换。"
    },
    {
      "title": "学习中心",
      "description": "提供诊断练习和评估。"
    }
  ]
}
```

### 示例3：完整配置

```json
{
  "platform": {
    "name": "儿童先心病智能诊断平台",
    "subtitle": "AI辅助先天性心脏病超声诊断",
    "tagline": "精准、高效、智能",
    "tagline2": "先心病辅助诊断工具"
  },
  "features": [
    {
      "title": "AI智能分析",
      "description": "采用深度学习模型自动识别心脏关键结构，生成热力图和诊断建议，辅助医师快速定位病灶。"
    },
    {
      "title": "多维判读",
      "description": "支持原始视频、热力图、边界框等多种模态视图，全面观察心脏功能状态，提升诊断准确性。"
    },
    {
      "title": "教育训练",
      "description": "提供标准化的诊断练习平台，实时评估诊断准确率和AI依赖程度，帮助医师持续提升技能。"
    }
  ],
  "buttons": {
    "topRight": "医师入口",
    "titleBelow": "点击登录"
  }
}
```

---

## 修改步骤

1. 打开 `UI/login_config.json` 文件
2. 修改对应的字段值
3. 保存文件
4. 刷新浏览器页面查看效果

---

## 注意事项

- JSON格式必须正确，不要遗漏逗号或引号
- 中英文均可使用
- 修改保存后可能需要清除浏览器缓存才能看到变化
- 如果配置加载失败，页面会显示部分默认值
